Octopress Setup

Jan 15th, 2012

Setting up Octopress has been quite easy and straightforward following the documentation. The only slight problem I found was with regard the deployment to github. The first step there for deploying with a Github user / organization is:

rake setup_github_pages

This script ask you for your Github repository url. From the documentation it could seem that they were referring to the Pages url http://albertovilla.github.com. If you use that as the URL you will find a strange error. The URL to be used here was: git@github.com:albertovilla/albertovilla.github.com.git

Apart from that small problem everything went fine during the installation.
